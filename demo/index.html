HotText

A UI library for slide controls. Controls can be dragged or clicked on, and will set the hidden input to which they are constrained. Works in major browsers and on touch devices!.

Javascript:

      

        
To initialize a single element:

        
new HotText(el);

        
To find all in page:

        
HotText.initializeAll();

      

      
Installation:

      

        
Just unzip into your web directory and link to the js/css files.
