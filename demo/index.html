HotText

A jQuery micro-plugin for slide controls. Controls can be dragged or clicked on, and will set the hidden select box to which they are constrained. Works in FF, Webkit, IE6+, Opera, and touch devices! Default distance on touch devices is 20px.

HTML:

Javascript:

None required with class hottext on init, alternatively...

$('div').hottext({ distance: 100 });

Installation:

Just unzip into your web directory and link to the js/css files.
